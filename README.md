# personal-pyscraper

Pyscraper es un web scraper desarrollado en python, la hice para uso personal para páginas de componentes de PC (Maximus y Compragamer por ahora).

El proyecto tiene 3 formas de mostrar la informacion por pantalla:
    1. La app que es una interfaz web hecha en React.
    2. La GUI que está obsoleta porque no soporta los filtros.
    3. Main que si se ejecuta de manera interactiva, se puede utilizar para extraer los json directamente, por linea de comandos.
